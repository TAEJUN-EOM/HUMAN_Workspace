이벤트 : 영향을 미치는 사건
이벤트 소스 (객체) : 이벤트가 발생된 객체
이벤트 핸들러 : 이벤트 처리하는 객체, 함수, 메서드
자바스크립트가 지원하는 이벤트 :
1.브라우저 또는 애플리케이션에서 스스로 발생하는 이벤트
2. 사용자가 정의하고 , 사용자가 발생시키는 이벤트
이벤트 종류 :
마우스 이벤트
키보드 이벤트
HTML 프레임 이벤트
HTML 입력 양식(form태그내에 input종류) 이벤트
구조 변화 이벤트
터치 이벤트
이벤트 이름 (Event Name) 또는 이벤트 타입 (Event Type ) : 예) load
이벤트 속성은 이벤트 핸들러 함수를 연결 : 예) onload = function(){}
이벤트 모델의 종류 :
1. DOM Level 0 - 고전 이벤트 모델, 인라인 이벤트 모델, 하나의 이벤트 핸들러만 가질 수 있음
2. DOM Level 2 - 표준 이벤트 모델, 여러 이벤트 핸들러 추가 가능
