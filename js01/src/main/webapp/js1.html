함수를 정의해서 필요할 때마다 호출해야 하는 경우 <head> 태그에 자바스크립로 정의합니다.
html페이지의 내용이 렌더링되면서 자바스크립트가 즉시 실행되도록 하려면 <body> 태그에 자바스크립로 정의합니다.
변수 선언 : var 변수명 = 초기값;
수명, 함수명, 객체명은 식별자 규칙에 유효해야 합니다.
자바 스크립트에서 문자열에 대한 표현은 "", ''를 사용합니다.
자바스크립트의 데이터 타입 : number, string, boolean, object, function, undefined
자바스크립트는 기본형 데이터는 정수와 실수를 구별하지 않음
자바스크립트는 배열을 객체로 취급함
자바스크립트는 객체지향언어+함수적 프로그래밍 언어
자바스크립트는 함수를 1급 객체로 취급함
