JavaScript 내장 객체 : Object, Array, String, JSON, Math,...
JavaScript Browser 객체 : window, document, location, screen,...

자바스크립트를 이용하면 브라우저의 정보에 접근하거나 브라우저의 여러 기능들을 제어할 수 있습니다.
브라우저 객체 모델(BOM)은 문서 객체 모델(DOM)과는 달리 W3C의 표준 객체 모델은 아닙니다.
window 객체 : 웹 브라우저의 창(window)을 나타내는 객체
window 객체의 메소드는 전역 함수이며, window 객체의 프로퍼티는 전역 변수가 됩니다.
window 객체의 innerHeight와 innerWidth 프로퍼티를 이용하면, 브라우저의 창 크기를 설정할 수 있습니다.
window 객체의 모든 메소드나 프로퍼티를 사용할 때는 window라는 접두사를 생략할 수 있습니다.
document 객체 : 브라우저 창에 표시되는 내용에 해당하는 문서(document)를 나타내는 객체
screen 객체 : 운영체제 화면의 속성을 가지는 객체 (height, width, availWidth, availHeight, colorDepth, pixcelDepth)
location 객체 : 브라우저의 주소 표시줄과 관련된 객체 (href, host, hostname, port, ..assign(url), reload(), replace(url))
location="http://www.naver.com";
location.href="http://www.naver.com";
location.assign( "http://www.naver.com");
location.replace("http://www.naver.com");
페이지 새로 고침 방법 :
location=location; location.href=location.href; location.assign(location); location.replace(location); location.reload();
