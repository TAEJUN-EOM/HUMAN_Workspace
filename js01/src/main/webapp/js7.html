자바스크립트 언어 : 인터프리터 언어, 함수적 프로그래밍 언어, 객체지향 언어
객체 :
1. JSON으로 정의되는 객체는 동적으로 속성, 메소드를 추가하거나, 제거할 수 있습니다.
2. function 으로 객체 정의
3. ES6에서는 class으로 객체를 정의
자바스크립트의 모든 객체는 Object를 자동으로 상속받습니다.
자바스크립트에서 생성자 함수로 정의되는 객체는 prototype이라는 속성 객체를 가집니다.
prototype 공간에 생성자 함수로 생성되는 객체의 메소드들을 정의하게 되면 메모리 공간을 절약할 수 있습니다.
생성자 함수로 객체를 생성할 때는 new 키워드를 사용합니다.
