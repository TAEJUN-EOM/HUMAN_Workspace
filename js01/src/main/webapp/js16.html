createElement(tagName) : 새로운 노드(태그)객체를 생성
createTextNode(text) : 텍스트 노드(태그의 내용)를 생성
parentNode.appendChild(node) : 객체에 노드를 연결합니다.
setAttribute(name, value) : 태그 객체의 속성을 설정
getAttribute(name) : 태그 객체의 속성값 반환
태그에 내용을 추가할때 : 태그객체.innerHTML 속성 사용
문서(태그) 객체의 스타일 변경 : style 속성과 css속성 사용
removeChild(child) : 문서 객체의 자식 노드를 제거
