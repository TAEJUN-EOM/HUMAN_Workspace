변수명, 함수명, 객체명 , 클래스명 식별자 naming 규칙 :
키워드 사용 불가
특수문자 _, $ 허용, 문자로 시작해야 함
두번째 문자부터 숫자 허용
공백 포함 안됨

형변환 :
문자열과 숫자의 덧셈 연산의 경우, 문자열로 자동 형변환되며
덧셈외의 연산자의 경우에는 number로 형변환됩니다.
NaN(Not a Number)는 자바스크립트에서 숫자로 표현, 처리 할 수 없는 값을 의미합니다.
number 타입의 값으로 명시적 형변환은 Number()함수를 사용할 수 있습니다.
boolean값은 number 타입의 값으로 형변환 가능
boolean 타입으로의 형변환은 Boolean()함수를 사용
boolean 타입으로의 형변환은 !! 연산자 사용
자바스크립트 연산자 : 산술연산자, 논리연산자, 비교연산자 등 자바의 연산자와 동일하게 지원
==연산자는 값을 비교합니다. === 연산자는 값과 타입을 비교합니다.
!=연산자는 값을 비교합니다. !== 연산자는 값과 타입을 비교합니다.
