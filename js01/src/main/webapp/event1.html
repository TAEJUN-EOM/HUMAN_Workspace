이벤트 모델의 종류 :
1. DOM Level 0 - 고전 이벤트 모델, 인라인 이벤트 모델, 하나의 이벤트 핸들러만 가질 수 있음
2. DOM Level 2 - 표준 이벤트 모델, 여러 이벤트 핸들러 추가 가능
고전 이벤트 모델에서 이벤트 핸들러 제거시 이벤트 소스 객체의 on이벤트 속성에 null을 할당
이벤트 핸들러 함수 내에서 this는 이벤트가 발생된 소스 객체를 의미합니다.
기본 이벤트(default 이벤트) : HTML 태그 중 이미 이벤트 핸들러 가지고 있고, ex)
