변수 선언 :
1. var 변수 선언은 한번 선언했음에도 다시 선언한 경우 에러가 발생하지 않음
유연한 변수 선언으로 테스트에는 편리할 수 있지만, 코드량이 많아지면 어디에서 어떻게 사용될지
파악하기 힘들고 값이 바뀔 우려가 있다
2. let 변수 선언은 재선언시에 오류 발생
3. const 선언된 변수는 변경할수 없는 상수 (재선언과 값 재할당 모두 오류 발생)
사용자로부터 입력 받기 :
html에서는 form 태그 내에 <input>, <textarea> , <select> 등을 사용
문자열이나 숫자를 입력받을때 자바스크립에서는 window.prompt()사용
boolean값을 입력받을때 자바스크립에서는 window.confirm()사용
제어문 :
if, else , nested if, 다중 if문
Quiz> 사용자로부터 정수(0~100사이)를 입력받아서 짝수 홀수 출력
삼항 연산자 => (조건문) : (조건식)? true일때 수행 문장 : false일때 수행하는 문장
조건식1 && 조건식2(실행문장2) => 조건식1이 true이면 조건식2(실행문장2)수행
조건식1 && 조건식2(실행문장2) => 조건식1이 false이면 조건식2(실행문장2)수행 안함
true && 문장(실행문장)
조건식1 || 조건식2(실행문장2) => 조건식1이 true이면 조건식2(실행문장2)수행 안함
조건식1 || 조건식2(실행문장2) => 조건식1이 false이면 조건식2(실행문장2)수행
false || 문장(실행문장)
switch문 : javascript에서는 범위 비교 연산이 가능합니다. (case에 범위비교 표현식 정의 가능)
Quiz>사용자로부터 1~100 사이의 점수를 입력받아서 A(>=90), B(>=80), C(>=70), D(>=60), F(<60) 등급을 판별을 switch구문을 사용하여 자바스크립트로 구현하시오
반복문 : for(카운트변수 초기화 ; 조건식 ; 증감식 ){ 반복 수행 문장; }
while(조건) {반복 수행 문장;}
do { 반복 수행 문장;}while(조건);
반복횟수를 알고 있을때 for문 사용
반복 수행 조건을 알고 있을때 while문 사용(선 조건체크, 후 수행)
최초 1번은 조건과 관계없이 수행후 조건을 체크해서 반복 수행여부를 결정 do~while문 사용(선 수행, 후 조건체크 )
무한루프 : while(true){} , for(; ;){}
for in문으로 순차적 처리
